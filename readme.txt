AJAX-powered Flickr gallery builder with a neat user interface.

== Description ==
You can select the images from Flickr which you would like to include in a gallery. 
After you publish the post a gallery consisting of thumbnails appears in it. When you click a thumbnail a full-size image
appears in a popup window.

Very flexible and easy to use!

== Installation ==

Use Plugins menu in the Wordpress Administration panel.
